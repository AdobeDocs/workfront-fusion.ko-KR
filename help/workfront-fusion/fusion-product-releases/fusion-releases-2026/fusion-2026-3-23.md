# Workfront Fusion 릴리스 활동: 2026년 3월 23일 주

이 페이지에서는 2026년 3월 23일이 있는 주에 Adobe Workfront Fusion에서 향상된 모든 기능에 대해 설명합니다.

최근 모든 변경 사항의 목록은 [Adobe Workfront Fusion 릴리스 활동](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md)을 참조하십시오.

Workfront Fusion의 최근 버그 수정 목록을 보려면 [Workfront 유지 관리 업데이트](https://experienceleague.adobe.com/ko/docs/workfront-known-issues/releases/current-updates) 페이지를 참조하고 Workfront Fusion 유지 관리 업데이트 레이블이 지정된 업데이트를 확인하십시오.

## 이제 Adobe Authenticator 커넥터에서 사용자 정의 API 호출(폴링) 모듈을 사용할 수 있습니다

특정 조건 또는 제한이 충족될 때까지 API 호출을 반복할 수 있도록 사용자 지정 API 호출 만들기(폴링) 모듈을 만들었습니다. 이 모듈에는 반복 API 호출을 구성할 수 있는 옵션이 포함되어 있습니다. 특정 조건이 충족될 때까지 API 호출을 실행하고, 특정 재시도 횟수를 설정하고, 재시도 사이의 시간을 구성할 수 있습니다.

자세한 내용은 문서 Adobe Authenticator 모듈에서 [사용자 지정 API 호출 만들기(폴링)](/help/workfront-fusion/references/apps-and-modules/adobe-connectors/adobe-authenticator-modules.md#make-a-custom-api-call-polling)를 참조하십시오.
