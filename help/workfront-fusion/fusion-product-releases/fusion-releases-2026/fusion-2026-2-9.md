# Workfront Fusion 릴리스 활동: 2026년 2월 9일 화요일 주간

이 페이지에서는 2026년 2월 9일 화요일이 있는 주에 Adobe Workfront Fusion에서 이루어진 모든 개선 사항을 설명합니다.

최근 모든 변경 사항의 목록은 [Adobe Workfront Fusion 릴리스 활동](/help/workfront-fusion/fusion-product-releases/fusion-release-activity.md)을 참조하십시오.

Workfront Fusion의 최근 버그 수정 목록을 보려면 [Workfront 유지 관리 업데이트](https://experienceleague.adobe.com/ko/docs/workfront-known-issues/releases/current-updates) 페이지를 참조하고 Workfront Fusion 유지 관리 업데이트 레이블이 지정된 업데이트를 확인하십시오.

## 시나리오에서 사용할 사용자 지정 JavaScript 함수 만들기

시나리오를 보다 유연하게 제어하고 제어하기 위해 시나리오에 사용할 수 있는 사용자 지정 JavaScript 함수 라이브러리를 만드는 기능이 추가되었습니다. Fusion의 [함수] 영역에서 사용자 정의 함수를 만들 수 있습니다. 그런 다음 이러한 함수를 Adobe App Builder 모듈 형태로 시나리오에 추가합니다.

사용자 지정 기능은 Adobe App Builder을 통해 작동하므로 이를 사용하려면 조직에 Adobe App Builder 라이선스가 있어야 합니다.

Fusion에서 사용자 지정 함수를 사용하는 방법에 대한 자세한 내용은 [사용자 지정 함수를 사용하여 데이터 매핑](/help/workfront-fusion/create-scenarios/map-data/map-using-custom-functions.md)을 참조하십시오.

## 불완전한 실행 가드레일로 업데이트

크고 복잡한 시나리오를 더 잘 지원하기 위해 불완전한 실행에 대한 가드레일을 업데이트했습니다. 이제 불완전한 실행 제한은 시나리오에 따라 다릅니다. Fusion은 시나리오당 최대 1GB 또는 100개의 불완전 실행을 저장할 수 있습니다. 어느 제한에 먼저 도달하든 관계없이 말이다.

이전에는 제한 사항이 팀을 기반으로 했습니다.

Fusion의 보호 기능에 대한 자세한 내용은 [Fusion 성능 보호 기능](/help/workfront-fusion/references/scenarios/fusion-performance-guardrails.md)을 참조하십시오.

## 새 Microsoft SharePoint Online > 파일 모듈 만들기

Microsoft SharePoint Online 커넥터에 새 파일 모듈 만들기가 추가되었습니다. 새로운 모듈은 대용량 파일로 작업할 때 더 나은 성능을 제공합니다.

이전에 사용 가능했던 파일 만들기 모듈의 이름이 파일 만들기 (기존)로 변경되었습니다. 이전 모듈은 계속 사용할 수 있지만 새 모듈을 사용하는 것이 좋습니다.

Microsoft SharePoint Online 모듈에 대한 자세한 내용은 [Microsoft SharePoint Online 모듈](/help/workfront-fusion/references/apps-and-modules/third-party-connectors/sharepoint-modules.md)을 참조하십시오.
