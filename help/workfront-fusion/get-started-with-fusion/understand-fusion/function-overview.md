# 함수 개요

항목을 매핑할 때 함수를 사용하여 단순 또는 복합 공식을 만들 수 있습니다. 사용 가능한 함수는 Excel 및 일부 프로그래밍 언어의 함수와 유사합니다.

* 일반적인 논리, 수학, 텍스트, 날짜 및 배열을 평가합니다.
* 텍스트를 대문자로 변환, 텍스트 트리밍, 날짜를 다른 형식으로 변환 등과 같은 항목 값의 조건부 논리 및 변환을 수행할 수 있도록 해 줍니다.
